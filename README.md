Zimbra S3-compatible store manager (Zm S3)
=========================================
Zm S3 is the ECS S3 compatible store extension for Zimbra Collaboration Open Source Edition.

## Building Java extension
### Requirement
* JDK 17 or newer.
* Apache Maven 3.5 or newer for Maven build.
